# Digital Entropy

Professional cybersecurity toolkit for CTF competitions and security research.

## Overview

Digital Entropy is a client-side toolkit featuring 25+ tools for encoding, encryption, hashing, and text manipulation. All operations run locally in your browser.

## Features

### Encoding Tools
- Base64 Encode/Decode
- Base32 Encode/Decode
- URL Encoder/Decoder
- HTML Entity Converter
- Hex Converter
- Binary Converter

### Encryption
- RSA Encryption/Decryption (512/1024/2048-bit)

## Installation

```bash
git clone https://github.com/HwyNe/HwyNe.github.io.git
cd HwyNe.github.io
```

Open `index.html` in your browser. No build process or dependencies required.

## Usage

1. Select a tool from the sidebar
2. Enter your input
3. Click the action button
4. Copy the result
