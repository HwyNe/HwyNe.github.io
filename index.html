MD5 Hash Generator

MD5 (Message Digest Algorithm 5) produces a 128-bit hash value. While not recommended for security purposes, it's still widely used for checksums and data integrity verification.

SHA-1 Hash Generator

SHA-1 produces a 160-bit (20-byte) hash value. Though considered cryptographically broken, it's still used in legacy systems and git commits.

SHA-256 Hash Generator

SHA-256 is part of the SHA-2 family and produces a 256-bit hash. Widely used in blockchain, SSL certificates, and secure password storage.

SHA-512 Hash Generator

SHA-512 produces a 512-bit hash value. Offers higher security than SHA-256 and is used in high-security applications.

Base64 Encoder / Decoder

Base64 encoding converts binary data to ASCII text format using A-Z, a-z, 0-9, +, /. Essential for data transmission and API calls.

AES Encryption / Decryption

Advanced Encryption Standard (AES) is a symmetric encryption algorithm. Industry standard for securing sensitive data.

RSA Encryption / Decryption

RSA is an asymmetric cryptographic algorithm using public/private key pairs. Foundation of modern secure communications.

URL Encoder / Decoder

Percent-encoding for safe URL transmission. Converts special characters to %XX format.

Hexadecimal Converter

Convert between text and hex (base-16) representation. Essential for low-level debugging and binary analysis.

Binary Converter

Convert text to binary (0s and 1s) and vice versa. Understand how computers store data at the lowest level.

Caesar Cipher

Classic substitution cipher that shifts each letter by a fixed number of positions. Named after Julius Caesar.

ROT13 Cipher

Special case of Caesar cipher with shift of 13. Applying ROT13 twice returns the original text.

Morse Code Translator

Convert text to Morse code (dots and dashes) and vice versa. Telegraph communication system invented in 1830s.

JSON Formatter & Validator

Format, minify, and validate JSON data. Essential for API development and debugging.

UUID Generator (v4)

Generate universally unique identifiers. Perfect for database keys and unique session IDs.

Secure Password Generator

Generate cryptographically strong passwords. Customize length and character types for maximum security.

QR Code Generator

Create QR codes for URLs, text, WiFi credentials, and more. Scannable by any smartphone camera.

Regular Expression Tester

Test and debug regex patterns. Real-time matching with flags support (g, i, m).

Word & Character Counter

Real-time statistics for your text. Perfect for essays, tweets, and content optimization.
